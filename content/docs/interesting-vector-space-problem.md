# An interesting vector space problem with \(\mathbb{R}^{+}\)

## Statement

{{< katex display=true >}}
\mathbb{R}^{+}
{{< /katex >}}

\[x\oplus y=xy\]

\[x,y\in\mathbb{R}^{+}\]

\[\lambda\odot x=x^{\lambda}\]

\[\lambda\in\mathbb{R}\]

## Proof

\[a=b^{\log_{b}\left(a\oplus b\right)-1}\]

\[a=\log_{b}\left(a\odot b\right)\]
